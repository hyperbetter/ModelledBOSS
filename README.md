# modelledBoss
仿BOSS直聘移动端APP
## 环境要求
需要NodeJS 8+环境
## 安装
* $ git init
* $ git clone https://github.com/hyperbetter/modelledBoss.git
* $ npm install
* 或者
* $ git init
* $ git clone git@github.com:hyperbetter/modelledBoss.git
* $ npm install
## 运行
$ npm start
服务器默认端口是5000，客户端默认端口是3000
## 功能简介
* 1.用户注册/登录功能
* 2.好友列表
* 3.消息列表
* 4.用户中心
